### 개발 목표
- 디저트 레시피를 찾을 때 pc와 모바일에서 편리하게 찾을 수 있는 Smooth한 웹앱 페이지 개발해보기
- 저당, 비건, 통밀 등 다양한 식문화를 가진 이색 디저트 레시피를 카테고리화 하여 수록하고, 맞춤형 서비스로도 추천할 수 있게 하기
- 사이드 프로젝트 <암기장 VOCA>를 배포하며 알게 된 문제점과 원인으로 예측되는 개념들을 사용해보기<br/>
  ```
  문제: netlify에 프로젝트 코드를 빌드하였는데 DB에 저장된 data들이 반영되지 않음
  
  해결 접근: 참고했던 리액트,  Node.js 책에서 데이터가 저장된 엔드포인트 url을 불러올 때에는
            axios와 express라는 개념이 함께 사용되는 것을 보고 새 프로젝트를 통해 두 개념들을 직접 사용해보며 익히고 해결하기로 목적 설정
  
  원인 추정 회고: https://chamypuppy.tistory.com/108
  ```

- 직접 사용하기의 목적 달성하기!💓

개발 노트

### 개발 기능
- 리액트와 node.js 그리고 MySQL을 연동하기
- node.js를 이용하여 필요한 DATA API를 만들기
- node.js와 카카오 로그인 API를 연동하기 (REST API로 개발)
- 세션과 쿠키를 사용하여 로그인과 로그아웃 상태 저장과 삭제

<br/>

### 개발 예정 기능
- 카카오 로그아웃 기능 해결(✔ 25.02.23 해결)
- 레시피 검색 기능 개발
- 레시피 CRUD 기능 개발
- 레시피 좋아요 기능과 모아보기 기능 개발
- 추천하고픈 레시피들 모음 페이지 개발
- 맞춤형 서비스 리서치
  회원가입시 내가 이 디저트 레시피 페이지에서 원하는 기능들을 리서치<br/>
  리서치 후 나의 선호도에 맞는 레시피를 우선적으로 보여줄 수 있게 맞춤형 서비스 반영하기
